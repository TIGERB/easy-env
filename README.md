# How to use?

Please install the docker & npm & yarn first!


> export EASY_PATH=$(pwd) && export PATH="$PATH:$EASY_PATH/bin"


### Commands:

```
Usage:
    easy init/start/restart/stop/destroy

Example:
    easy init
    easy start
    easy restart
    easy stop
    easy destroy
```
